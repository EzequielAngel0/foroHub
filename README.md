# ForoHub

ForoHub es un proyecto de pila completa para foros con autenticacion JWT, gestion de topicos y frontend en React.

## Inicio rapido

1. Backend

```bash
./mvnw spring-boot:run
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```

- Backend: Java 17, Spring Boot 4, Spring Security, Spring Data JPA, Flyway, MySQL
- Frontend: React, Vite, utilidades Tailwind, Axios
